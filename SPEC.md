# 台股當沖/隔日沖監測系統 - 規格書

## 1. 專案概述

**專案名稱**: TaiwanStockScanner  
**類型**: 響應式 Web 應用 (React + Node.js)  
**核心功能**: 即時監測台股當沖/隔日沖標的，透過技術指標篩選強勢股  
**目標用戶**: 投資小白、當沖/隔日沖交易者

---

## 2. UI/UX 規格

### 2.1 整體風格
- **設計語言**: 現代簡約金融風格
- **配色方案**:
  - 主色: `#1a1a2e` (深藍黑)
  - 次色: `#16213e` (深藍)
  - 強調色: `#e94560` (漲停紅), `#00d9a5` (買進綠)
  - 文字: `#eaeaea` (淺灰白), `#a0a0a0` (次要灰)
  - 背景: `#0f0f1a` (近黑)
- **字體**: Inter, sans-serif
- **圓角**: 12px (卡片), 8px (按鈕)

### 2.2 響應式斷點
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### 2.3 頁面結構

#### Header 區
- Logo + 系統名稱
- 即時大盤指數 (加權指數 + 漲跌)
- 當日日期時間

#### 主內容區 (3欄 Desktop, 2欄 Tablet, 1欄 Mobile)
1. **篩選條件面板** (左側)
   - 漲跌幅區間 (滑桿)
   - 成交量門檻 (輸入框)
   - 價格區間 (高低)
   - 技術指標勾選 (KD、MACD、RSI)
   - 產業別篩選 (下拉選單)
   - 「開始監測」按鈕

2. **即時監測表格** (中央 - 主體)
   - 欄位: 股票代碼/名稱、現在價格、漲跌幅、成交量、當日高低、買賣力
   - 支援排序功能
   - 點擊展開詳細資訊
   - 漲跌幅用顏色區分 (紅漲/綠跌)

3. **選股詳情面板** (右側)
   - 選擇股票後顯示詳細資訊
   - 走勢圖 (簡易折線圖)
   - 技術指標數值
   - 快速下單按鈕 (預留)

#### Footer 區
- 監控狀態 (連線中/已連線)
- 最後更新時間

### 2.4 動畫與互動
- 價格變動時閃爍效果 (綠漲/紅跌)
- 資料載入時 Skeleton loading
- 卡片 hover 時輕微上浮 + 陰影加深
- 篩選條件變更時平滑過渡

## 3. 功能規格

### 3.1 核心功能

#### 即時行情監測
- 透過 WebSocket 接收即時報價
- 模擬資料源 (可替換為真實 API)
- 價格/漲幅/成交量即時更新

#### 股票篩選引擎
- 漲跌幅範圍: -10% ~ +10%
- 成交量門檻: 1000~1000000 股
- 價格範圍: $10~$1000
- 技術指標篩選:
  - KD黃金交叉 (K>D且D<20)
  - MACD柱狀體正值
  - RSI > 50

#### 監測名單管理
- 新增/刪除監測股票
- 自訂警示價格
- 歷史紀錄追蹤

### 3.2 資料模型

```typescript
interface Stock {
  code: string;           // 股票代碼
  name: string;           // 股票名稱
  price: number;         // 現價
  change: number;        // 漲跌
  changePercent: number; // 漲跌幅%
  volume: number;        // 成交量
  high: number;          // 當日最高
  low: number;           // 當日最低
  turnover: number;      // 成交金額
  kValue: number;        // KD K值
  dValue: number;        // KD D值
  macd: number;          // MACD
  rsi: number;           // RSI
  industry: string;      // 產業別
  lastUpdate: string;    // 最後更新時間
}
```

## 4. 技術堆疊

- **前端**: React 18 + Vite + Tailwind CSS
- **圖表**: Recharts
- **後端**: Node.js + Express + Socket.io
- **模擬資料**: 內建假資料產生器
- **部署**: 可打包為靜態網頁
